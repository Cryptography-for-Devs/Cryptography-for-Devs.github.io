This is Cryptography for Developers

Made by @vkweb. Previously, freeCodeCamp.org curriculum contributor.

Learn those ignored but important concepts with this free interactive course.

  • Symmetric key encryption
  • Rainbow tables, salt
  • Password based key derivation function (PBKDF-2)
  • Cryptographic hash functions
  • Password hashing
