0. Before we start

Welcome! this course explains the intuitions behind the cryptographic techniques that come into daily lives of developers.

Before we move forward, let us make one thing clear.

You must never ever implement your own cryptographic algorithm in production. Because implementing these algorithms is the task of cryptography experts. So, we developers must always use the libraries our framework or language provides.
